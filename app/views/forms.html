For the floating labels to work properly without any javascript we need to:
  1. add "float-label" class to the "field-group" wrapper
  2. The label tag must be insert after the input field with class "placeholder"
  3. the input/textarea element must have required attribute
							
<div class="field-group float-label">
	<input type="text" id="demo-input-1" required pattern=".{1,}">
	<label class="placeholder" for="demo-input-1">This is a label</label>
	<p class="help">This is a contextual help for the field above.</p>
</div>
Radio buttons and checkboxes
If you want to use the custom Radio/Checkbox Inputs you need to put the <label> after the <input> 
							
<input type="radio" name="demo-radio" id="demo-radio-1">
<label for="demo-radio-1">Radio 1</label>

<input type="radio" name="demo-radio" id="demo-radio-2">
<label for="demo-radio-2">Radio 2</label>
